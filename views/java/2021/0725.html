                          # Java标识符（凡是程序员自己有权利命名的单词都是标识符）

                          • 但是main是一个方法的名称是一个标识符，但不能修改，是语言发明者定义的
                          • 标识符可以标识：类名，方法名，变量名，接口名，常量名。。。

                          # 标识符的命名规则

                          • 标识符只能由数组，字母，下划线，$,不能含有其他符号
                          • 标识符不能以数字开头
                          • 关键字不能做标识符（public,class,void,static）
                          • 标识符严格区分大小写

                          # 变量

                          • 变量其实就是内存当中存储数据的最基本单元。
                          • 在java语言中任何数据都是有数据类型的，例如整数型：int。不同的数据类型，在内存中分配的空间不同。
                          • 变量三要素：变量的类型，名字，和保存的值。

                          # 变量的声明和定义

                          • 数据类型 变量名 例如：int nianLing;
                          • 注意，方法体当中的代码遵循自上而下的顺序依此逐行执行。
                          • 在同一个域中变量名不能重复声明（跟类型无关）。

                          # 变量的分类

                          • 在方法体中声明的变量成为局部变量，在类体内声明的变量成为成员变量。 局部变量只在方法体当中有效，方法体执行结束，该变量内存也就释放了。

                          # 变量的作用域

                          • 什么是作用域：变量的有效范围。
                          • 记住：变量作用域，出了大括号就不认识了。
                          • java中很重要的原则：就近原则

                          # 数据类型

                          • 基本数据类型：整数型（byte,short int long）,浮点型（float,double）,布尔型（boolean）,字符型（char）
                          • 引用数据类型：字符串型string属于引用数据类型。除了基本数据类型，都是引用数据类型

                          # 计算机存储单位

                          • 1字节 = 8bit =>1byte = 8bit; 1bit就是1或0
                          • 1k=1024byte
                          • 基本数据类型占用字节数量,byte:1;short:2;int:4;long:8; float:4;double:8;boolean:1;char:2; byte是[-128~127] short是[-32768~32767] char是[0~65535] 在计算机中最左边位表示符号位，0表示正，1表示负 short和char实际容量相同，但char可以表示更大的数字。char表示的是文字，没有正负之分。

                          # 字符编码

                          ASCII：'a'-->97;'A'-->65;'0'-->48 ASCII ISO-8859-1(latin-1) GB2312 GBK GB18030 Big5 繁体中文 unicode(utf-8,utf-16,utf-32)

                          # 转义字符

                          • java语言中''负责转义；\的出现会将\后面的相邻字符进行转义；
                          • '\'两个反斜杠代表一个普通的\

                          # 整型

                          • 在java中，整数型的字面量默认被当作int类型处理，如果希望该字面量被当作 long类型来处理，需要在字面量后面加上"L/l"
                          • 小容量可以直接转化位大容量类型，叫自动类型转化
                          • 大容量转化成小容量，要想编译通过，需要添加强制类型转换符[(int)100]，但运行时可能损失精度

                          # 二进制原码，反码，补码

                          • 对于正数：二进制原码，反码，补码都是它本身
                          • 对于负数：反码-->符号位不变，其余求反；补码为反码+1

                          # 浮点型

                          • float 单精度；double 双精度；
                          • 在java中，如果在银行或者财务方面，double也是远远不够，java提供了一种精度更高的类型 java.math.BigDecimal.
                          • 在float和double存储数据时，存储的都是近似值
                          • 任意一个浮点型存储的空间都比整数型空间大得多
                          • java中规定任意一个浮点型数据默认当作double类型来处理，如果想让这个浮点型被当作float类型处理 那么在后面添加'F/f'。

                          # 基本类型转化

                          • 八种数据类型，除了boolean类型不能转化，剩余七种类型之间都可以进行转化。
                          • 如果整型字面量没有超出byte，short, char范围，可以直接将其赋值。
                          • 容量排序：byte< short(char) < int < long < float < double .小到大转化成为自动转化
                          • 大容量转化成小容量需要强制转化。
                          • byte,short,char混合运算时，先转化成int类型再运算。
                          • 多种混合类型运算时，先自动转化成容量最大的再做运算

                          # 运算符

                          • 算术运算符：+ - * / ++ --
                          • 关系运算符有：> < = != >= <= (运算结果一定是布尔型)
                          • 逻辑运算符： &(与) |(或) !(非) &&(短路与) ||(短路或) true & true -->true (一假为假) 或（一真为真）
                          • 赋值运算符：= += -= *= /= 很重要的语法机制：使用扩展运算符永远不会改变运算结果类型
                          • 条件运算符(三目运算符):布尔表达式？表达式1：表达式2
                          • 字符串连接运算符：+ 当加号运算符两边都是数字类型时，求和，当字符串任意一边是字符串类型时是字符串拼接。

                          # 选择语句

                          • if: 是条件语句

                          • 第一种 if(布尔表达式){java语句;}

                          • 第二种 if(布尔表达式){java语句;}else{java语句;}

                          • 第三种 if(布尔表达式){java语句;}else if(布尔表达式1){java语句;} ...

                          • 第四种 if(布尔表达式){java语句;}else if(布尔表达式1){java语句;} ...else{java语句;}

                          • 注意，对于if来说，任何情况下只能有一个分支执行，只要有一个分支执行了，整个if语句就结束了。

                          • 当分支当中，java语句只要一条时，大括号可以省略，一般为了可读性不省略。
