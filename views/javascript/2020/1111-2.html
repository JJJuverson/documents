    # 同步和异步

    • 同步阻塞：A调用B，B处理获得结果，才返回给A。A在这个过程中，一直等待B的处理结果，拿到结果之后才会再去做下边的事。
    • 异步非阻塞：A调用B，无需等待B的结果，B通过状态，通知等来通知A或回调函数函数来处理。
